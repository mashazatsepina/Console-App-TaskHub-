# TaskHub — Менеджер задач

Консольное приложение для управления задачами, написанное на C# (.NET 8).

## Возможности

- Создание задач с названием, описанием, приоритетом, дедлайном и статусом
- Просмотр всех задач, выполненных, невыполненных и задач с высоким приоритетом
- Редактирование и удаление задач
- Поиск по названию, статусу и приоритету
- Статистика: общее количество, выполненные, просроченные, распределение по приоритетам
- Сохранение и загрузка задач в формате JSON
- Фоновая проверка дедлайнов с уведомлениями в консоль

## Запуск

```bash
dotnet run
```
